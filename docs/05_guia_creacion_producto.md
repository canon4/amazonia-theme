# Guía de Creación de Producto por Rol

Esta guía detalla paso a paso cómo crear un producto en el marketplace. Principalmente es una tarea del **Vendedor**, aunque el **Administrador** puede editar o crear productos a nombre de cualquier vendedor si es necesario.

## Proceso de Creación (Vendedor)

1. Ingresa a tu panel de Vendedor (Dashboard).
2. En el menú lateral, selecciona **Productos**.
3. Haz clic en el botón **Añadir Nuevo**.
4. Selecciona el tipo de producto (usualmente *Producto Simple* o *Producto Variable* si tiene tallas/colores).

## Explicación de Campos (Qué va en cada campo)

A continuación, se detalla qué información exacta debe introducirse en cada uno de los campos principales para mantener un estándar de alta calidad en todo el marketplace:

### Datos Básicos
- **Título del Producto:** El nombre público. *Debe ser claro y descriptivo. (Ej. "Zapatillas de Running Nike Air Zoom")*
- **Precio Normal:** El precio de venta estándar (en la moneda local, sin comas para miles, usando punto para decimales).
- **Precio Rebajado:** (Opcional) Si el producto está en oferta.
- **Descripción Corta:** Un resumen de 2 a 3 líneas que aparece justo al lado de la imagen. *Destaca el principal beneficio o característica.*
- **Descripción:** Texto largo que va en la parte inferior. *Incluye detalles técnicos, materiales, garantías, y recomendaciones de uso.*
- **Imágenes:**
  - *Imagen principal:* Una foto limpia (preferiblemente con fondo blanco o neutro) del producto.
  - *Galería de imágenes:* Fotos adicionales desde distintos ángulos o en contexto de uso.

### Inventario
- **SKU:** (Código de Referencia). *Un identificador interno único para ti. (Ej. ZAP-NIK-001)*.
- **Gestión de Stock:** Marca la casilla para indicar cuántas unidades tienes físicamente disponibles.
- **Cantidad en Stock:** Número de unidades. *El sistema restará 1 automáticamente por cada venta.*

### Envío
- **Peso:** Cuánto pesa el paquete (en kg o gramos, según configuración). *Es crucial para calcular el costo de envío.*
- **Dimensiones:** Largo, ancho y alto del paquete empaquetado.
- **Clase de Envío:** Selecciona un perfil de envío si ofreces diferentes tarifas (ej. "Envío Frágil").

### Atributos y Categorías
- **Categorías:** Elige el sector correcto del catálogo del marketplace (ej. "Ropa -> Zapatos"). *No elijas más de 2 categorías para no afectar el SEO.*
- **Etiquetas (Tags):** Palabras clave por las que el cliente podría buscar tu producto (ej. "deporte, correr, nike").
- **Atributos (Para productos variables):** Si el producto tiene opciones, aquí defines el "Color" o la "Talla". Luego, en la pestaña *Variaciones*, creas las combinaciones (ej. Azul - Talla 42).

## Flujo de Aprobación

- **Vendedores de Confianza:** Al guardar, el producto pasa al estado **Publicado** de inmediato.
- **Nuevos Vendedores / Control Estricto:** Al guardar, el producto queda en estado **Pendiente de Revisión**. El Administrador recibirá una notificación, lo revisará bajo las políticas de calidad y lo publicará o lo devolverá a Borrador con comentarios.
